Gitnalysis - A Github Analysis Application
© 2015 Caleb Rogers

Gitnalysis is a Github analytics tool. It allows users to search for topics that interest them, be they technical or non-technical topics. Results from searches are curated to allow users to find Github Users, Repos, and Organizations related to their chosen topic of interest.

Clicking into a given repo will allow users to see issues, sorted by difficulty. We used the Google Predict API, an accessible Machine Learning API, to train a model what an "beginner," "intermediate," or "advanced" issue look like.
