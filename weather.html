This is an example use case of the Electric Imp Env Sensor Tail module. The module is a standalone board with attachments for measuring ambient light level, air pressure, humidity, and temperature. It also contains an LED. All components are controllable via the Electric Imp API, and the device used below is connected via WIFI in the Electric Imp offices.
